HTML

HTML (HyperText Markup Language) adalah bahasa markup standar yang menjadi kerangka dasar untuk membangun sebuah website. HTML mendefinisikan struktur dan makna dari konten web, mulai dari teks, gambar, hingga tautan.

Bahasa markup atau bahasa markah adalah sistem yang menggunakan anotasi atau kode untuk menjelaskan struktur, format, atau makna dari sebuah dokumen teks. Dengan cara ini, perangkat lunak dapat memproses dan menyajikan informasi tersebut secara tepat kepada pengguna. Contoh bahasa markup yang umum digunakan adalah HTML dan Markdown.

Dalam praktiknya, HTML tersusun atas elemen-elemen yang memberikan instruksi kepada browser mengenai cara menampilkan konten. Elemen-elemen ini juga berfungsi memberi label pada bagian tertentu dari halaman, misalnya teks sebagai judul, paragraf, atau tautan.

HTML menggunakan markup berupa tag untuk membedakan elemen dari teks biasa. Tag ditulis di dalam tanda kurung sudut < > dan tidak bersifat peka huruf besar-kecil (case-insensitive), meskipun praktik terbaik adalah menuliskannya dalam huruf kecil. Beberapa contoh elemen HTML yang umum digunakan antara lain:

  • <h1> untuk heading
  • <p> untuk paragraf
  • <a> untuk tautan
  • <img> untuk gambar
  • <ul> dan <li> untuk daftar
